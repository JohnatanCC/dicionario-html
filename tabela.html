Tags de Tabela

<table>

Define uma tabela.


<table>

  <tr>

    <th>Cabeçalho 1</th>

    <th>Cabeçalho 2</th>

  </tr>

  <tr>

    <td>Dado 1</td>

    <td>Dado 2</td>

  </tr>

</table>

<tr>

Define uma linha em uma tabela.


<tr>

  <td>Dado 1</td>

  <td>Dado 2</td>

</tr>

<td>

Define uma célula de dados em uma tabela.


<tr>

  <td>Dado 1</td>

  <td>Dado 2</td>

</tr>

<th>

Define uma célula de cabeçalho em uma tabela.


<tr>

  <th>Cabeçalho 1</th>

  <th>Cabeçalho 2</th>

</tr>

<thead>

Define um grupo de cabeçalhos em uma tabela.


<thead>

  <tr>

    <th>Cabeçalho 1</th>

    <th>Cabeçalho 2</th>

  </tr>

</thead>

<tbody>

Define o corpo de uma tabela.


<tbody>

  <tr>

    <td>Dado 1</td>

    <td>Dado 2</td>

  </tr>

</tbody>

<tfoot>

Define o rodapé de uma tabela.


<tfoot>

  <tr>

    <td>Rodapé 1</td>

    <td>Rodapé 2</td>

  </tr>

</tfoot>

<caption>

Define uma legenda para a tabela.


<table>

  <caption>Legenda da Tabela</caption>

  <tr>

    <th>Cabeçalho 1</th>

    <th>Cabeçalho 2</th>

  </tr>

  <tr>

    <td>Dado 1</td>

    <td>Dado 2</td>

  </tr>

</table>

<col>

Define propriedades de coluna para cada coluna dentro de um grupo de colunas.


<colgroup>

  <col span="2" style="background-color: #f2f2f2">

</colgroup>

<colgroup>

Especifica um grupo de uma ou mais colunas em uma tabela para formatação.


<table>

  <colgroup>

    <col span="2" style="background-color: #f2f2f2">

  </colgroup>

  <tr>

    <th>Cabeçalho 1</th>

    <th>Cabeçalho 2</th>

  </tr>

  <tr>

    <td>Dado 1</td>

    <td>Dado 2</td>

  </tr>

</table>
