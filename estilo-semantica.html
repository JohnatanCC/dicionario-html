Tags de Estilo e Semântica

<style>

Define estilo CSS para um documento HTML.


<style>

  body {

    background-color: lightblue;

  }

</style>

<div>

Define uma seção genérica em um documento.


<div>Este é um div.</div>

<span>

Define uma seção pequena e genérica dentro de um documento.


<span>Este é um span.</span>

<header>

Define um cabeçalho para um documento ou seção.


<header>

  <h1>Cabeçalho</h1>

</header>

<footer>

Define um rodapé para um documento ou seção.


<footer>

  <p>Rodapé</p>

</footer>

<main>

Define o conteúdo principal de um documento.


<main>

  <p>Conteúdo principal.</p>

</main>

<section>

Define uma seção em um documento.


<section>

  <h2>Seção</h2>

  <p>Conteúdo da seção.</p>

</section>

<article>

Define um conteúdo independente em um documento.


<article>

  <h2>Artigo</h2>

  <p>Conteúdo do artigo.</p>

</article>

<aside>

Define conteúdo lateral relacionado ao conteúdo principal.


<aside>

  <p>Conteúdo lateral.</p>

</aside>

<nav>

Define uma seção de links de navegação.


<nav>

  <ul>

    <li><a href="#home">Home</a></li>

    <li><a href="#about">About</a></li>

  </ul>

</nav>

<figure>

Define conteúdo autônomo, como imagens, diagramas, etc.


<figure>

  <img src="imagem.jpg" alt="Descrição da imagem">

  <figcaption>Legenda da imagem</figcaption>

</figure>

<figcaption>

Define uma legenda para o elemento <figure>.


<figure>

  <img src="imagem.jpg" alt="Descrição da imagem">

  <figcaption>Legenda da imagem</figcaption>

</figure>

<details>

Define detalhes adicionais que o usuário pode visualizar ou ocultar.


<details>

  <summary>Resumo</summary>

  <p>Detalhes adicionais visíveis ao expandir.</p>

</details>

<summary>

Define um título para o elemento <details>.


<details>

  <summary>Resumo</summary>

  <p>Detalhes adicionais visíveis ao expandir.</p>

</details>

<dialog>

Define uma caixa de diálogo ou janela.


<dialog>

  <p>Esta é uma caixa de diálogo.</p>

</dialog>
