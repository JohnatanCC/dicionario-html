Tags de Estrutura

<html>

Define a estrutura básica de um documento HTML.


<html lang="pt-BR">

<head>

<!-- Cabeçalho do documento -->

</head>

<body>

<!-- Conteúdo do documento -->

</body>

</html>

<head>

Contém metadados/informações para o documento.


<head>

<meta charset="UTF-8">

<title>Título do Documento</title>

</head>

<base>

Especifica a URL base para todas as URLs relativas em um documento.


<base href="https://www.exemplo.com/">

<body>

Define o corpo do documento.


<body>

<h1>Título Principal</h1>

<p>Parágrafo de exemplo.</p>

</body>

<title>

Define o título do documento que aparece na aba do navegador.


<title>Meu Site</title>

<meta>

Define metadados sobre o documento HTML.


<meta charset="UTF-8">
