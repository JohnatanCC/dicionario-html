Introdução

Bem-vindo ao dicionário de tags HTML. Aqui você encontrará informações sobre diversas tags HTML, suas funcionalidades e exemplos de uso.

A estrutura básica de um documento HTML consiste nas seguintes tags principais:

  • <!DOCTYPE html>: Declara o tipo de documento e a versão do HTML.
  • <html>: Define a raiz de um documento HTML.
  • <head>: Contém metadados, links para scripts e folhas de estilo.
  • <meta>: Define metadados sobre o documento, como charset e viewport.
  • <link>: Usado para linkar folhas de estilo externas.
  • <title>: Define o título do documento, que é exibido na aba do navegador.
  • <body>: Contém todo o conteúdo visível da página.

Aqui está um exemplo básico de um documento HTML:

<!DOCTYPE html>
<html lang="pt-BR">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Título do Documento</title>
        <link rel="stylesheet" href="styles.css">
    </head>
    <body>
        <p>Conteúdo da página.</p>
    </body>
</html>

Resumo das Seções

Tags de Estrutura

As tags de estrutura são usadas para definir a estrutura básica de um documento HTML. Elas incluem:

  • <html>: Define a raiz de um documento HTML.
  • <head>: Contém metadados, links para scripts e folhas de estilo.
  • <body>: Contém todo o conteúdo visível da página.
  • <header>: Define uma seção de cabeçalho para um documento ou uma seção.
  • <nav>: Define um conjunto de links de navegação.
  • <section>: Define uma seção em um documento.
  • <article>: Define conteúdo independente e de auto-contido.
  • <aside>: Define conteúdo lateral que é relacionado ao conteúdo principal.
  • <footer>: Define um rodapé para um documento ou uma seção.

Tags de Formulário

As tags de formulário são usadas para criar formulários de entrada de dados em HTML. Elas incluem:

  • <form>: Define um formulário HTML para entrada de dados do usuário.
  • <input>: Define um campo de entrada de dados.
  • <textarea>: Define uma área de texto de várias linhas.
  • <button>: Define um botão clicável.
  • <select>: Define uma lista suspensa.
  • <option>: Define uma opção em uma lista suspensa.
  • <label>: Define um rótulo para um elemento de formulário.
  • <fieldset>: Define um grupo de elementos de formulário.
  • <legend>: Define uma legenda para um <fieldset>.

Tags de Mídia

As tags de mídia são usadas para incorporar conteúdo multimídia como áudio, vídeo e imagens. Elas incluem:

  • <audio>: Define conteúdo de som.
  • <video>: Define conteúdo de vídeo.
  • <img>: Define uma imagem.
  • <picture>: Define um container para múltiplas imagens.
  • <figure>: Define uma figura.
  • <figcaption>: Define uma legenda para uma figura.

Outras Tags

Além das tags mencionadas, existem várias outras tags úteis em HTML. Elas incluem:

  • <a>: Define um hyperlink.
  • <div>: Define uma seção ou divisão no documento HTML.
  • <span>: Define uma seção ou divisão em linha no documento HTML.
  • <br>: Insere uma quebra de linha.
  • <hr>: Insere uma linha horizontal.
