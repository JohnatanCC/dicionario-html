Tags de Formulário

<form>

Define um formulário HTML para entrada de dados do usuário.

<form>
    <input type="text" name="nome">
    <input type="submit" value="Enviar">
</form>

<input>

Define um campo de entrada de dados.

<input type="text" name="nome">

<textarea>

Define uma área de texto de várias linhas.

<textarea name="mensagem" rows="4" cols="50"></textarea>

<button>

Define um botão clicável.

<button type="button">Clique Aqui</button>

<select> e <option>

Define uma lista suspensa.

<select name="opcoes">
    <option value="opcao1">Opção 1</option>
    <option value="opcao2">Opção 2</option>
</select>

<label>

Define um rótulo para um elemento de formulário.

<label for="nome">Nome:</label>
<input type="text" id="nome" name="nome">

<fieldset> e <legend>

Define um grupo de elementos de formulário e uma legenda para o grupo.

<fieldset>
    <legend>Informações Pessoais</legend>
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome">
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
</fieldset>

<legend>

Define uma legenda para um <fieldset>.

<legend>Informações Pessoais</legend>
